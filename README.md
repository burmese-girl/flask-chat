### Run the App Locally

1. Clone the repository:

```bash
git clone https://github.com/burmese-girl/flask-chat.git
```

2. Navigate to the project directory:

```bash
cd flask-chat
```

3. Install the dependencies:

```bash
pip3 install -r requirements.txt
```

4. Run the app:

```bash
python3 app.py
```
